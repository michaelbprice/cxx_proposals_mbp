Generalized fold expressions

I. Introduction

II. Motivation

  // Generalized fold expression over template parameter packs (as proposed by N????)
  //
  template <typename T...>
  T sum (T... && args) { return args + ...; }

  // Generalized fold expression over a container
  //
  template <typename C>
  C::value_type sum (C && c) { return c + ...; }

  // Equivalent to:
  template <typename C>
  C::value_type sum (C && c)
  {
      C::value_type s{}; // default-initialized
      for (auto && e : c)
      {
           s += e;
      }
      return s;
  }

  //Essentially, it would expand a container, whether or not it was a constexpr or runtime entity.
  // Could be applied to standard containers, tuples, parameter packs, lists of reflected metadata, etc...

IV. Technical Specification

This paper is looking for guidance on whether this approach should be pursued. It provides no wording for the standard.
